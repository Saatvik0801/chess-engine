# Chess Engine in Java

## Overview
A console-based chess engine written in Java using object-oriented design principles.

## Features
- Polymorphic piece hierarchy
- Legal move validation
- Modular design

## Design Decisions
- Abstract base class for chess pieces
- Board represented as 2D array
- Move validation separated from piece logic

## How to Run
javac src/Main.java  
java src/Main
